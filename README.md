MERN AI Flow Application(Name:AIFlOWBUILDER)

A MERN application builds with React Flow  to implement  AI-powered text generation using OpenRouter API. Built with TypeScript, MongoDB, Express, React, and Node.js.

Features

-  Interactive flow chart using React Flow
-  AI-powered text generation with OpenRouter API (openrouter/auto)
-  Save conversations to MongoDB database
-  View conversation history (one Button on Dashboard with ICON)
-  TypeScript for type safety
- Modern UI with Tailwind CSS
- Real-time updates with React hooks(effect and state , memoization used  with usecallbacks to avoid expenive calculation and delay of response)

  Tech Stack

- Frontend: React, TypeScript, React Flow, Tailwind CSS
- Backend: Node.js, Express, TypeScript
- Database: MongoDB with Mongoose (Atlas)
- AI API: OpenRouter (openrouter/auto)
- Deployment: Render.com(Backend) and Vercel (Frontend)
